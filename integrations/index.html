Prerequisites

Environment setup​

  1. Partners should prepare at least 2 environments (development, production)
  2. Go to Portal to get clientId for your Game/Application

note

The development version for Portal is available at Portal Dev

The official version Portal would be released soon

Game Client​

Partners shall need to setup their client in accordance with the game engine, using the clientId obtained from our Portal

Unity​

Install Mirai SDK Package​

  • Install Package Dependencies

    • com.unity.nuget.newtonsoft-json
    • com.itisnajim.socketiounity

  • Go to Window -> Package Manager -> (+) Add from git:

    • https://github.com/mirailabs-co/mirai-sdk-unity.git#upm

Config​

Mirai SDK Settings window will automatically open if there is no clientId inserted, or else partner can go to Window -> Mirai SDK Settings.

  • Select Environment and insert clientId under
  • Type AppSchema for deeplink callback

Click "Apply Setting" to save.

Initialize​

If partners want to automatically init when start game, tick on "Auto Init".

Else, untick and init manualy using the following code:

await MiraiSDK.InitAsync();

JavaScript​

UDM File Download​

UDM File 
window.ShardsTech = {
  Connection, // connect websocket
  Core, // function of shards tech
};

Npm install (private)​

npm install @mirailabs-co/shards-tech

yarn add @mirailabs-co/shards-tech

Unreal Engine​

Before you begin​

  • Verify your Unreal Version is Unreal Engine 4.27, 5, or higher.
  • Ensure you have Visual Studio (or Rider) installed; it should be because Unreal Engine depends on it. In order for Unreal Engine to work properly you might need to install the following packages from the Visual Studio Installer: Desktop development with C++, Game development with C++, .NET desktop development.

Download and import the SDK into Project​

  1. In case you don't have an existing project, create a new project
  2. Close the Unreal editor.
  3. Create a directory in your project folder and name it Plugins.
  4. Download and import the ShardsTech plugin into your project.
  5. Open your project in Unreal Engine.
  6. You successfully imported the ShardsTech plugin.

Game Server​

Partners shall need their own server and provide us some APIs below:

API Authentication (Require)​

To authenticate players, you need to provide a user authentication API (you can configure this in the portal page) according to the following standard

Method: GET​

Endpoint: YOUR_URL_TO_AUTHEN​

Headers​

  • x-api-key: YOUR_API_KEY
  • Authorization: USER_TOKEN

Response​

  • If true
{
  "status": true,
  "data": {
    "userId": "string",
    "clientId": "string", //optional
    "name": "string", //optional
    "email": "string" //optional
  }
}
  • If false
{
  "status": false
}
