Full Integration

This integration requires the most effort from our partners, as it involves both integrating SDK features and designing the UI/UX within their own game. However, this approach grants them full control over the final product, allowing to meet their specific requirements and standards.

Process​

Step 1: Partner create account in self-integration portal​

Refer to the Environment setup section for more details

Step 2: Install SDK package base on game engine​

Refer to the Game Client section for more details

Step 3: Game server setup​

Refer to the Game Server section for more details

Step 4: Integrate SDK features​

Refer to the SDK section

Step 5: Design guild management UI/UX inside of the game​

Partner can freely design a full flow UI/UX that make use of the integrated SDK features
