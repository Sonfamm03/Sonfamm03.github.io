Partial Integration

This integration process enables partners to fully manage their guild management front-end by utilizing our provided UI kit. The UI kit is equipped with all the necessary SDK features, ensuring partners can easily adapt the front-end to meet their specific needs while leveraging the extensive functionalities already built into the kit.

Process​

Step 1: Create account in self-integration portal​

Refer to the Environment setup section for more details

Step 2: Install SDK package base on game engine​

Refer to the Game Client section for more details

Step 3: Set up game server​

Refer to the Game Server section for more details

Step 4: Integrate UI Kit React​

We provide a game UI kit with full features of Shards Tech on the client side (React web). With simple syntax, the integration testing process will be completed quickly.

Installation​

npm i @mirailabs-co/ui-kit-shards-tech

yarn add @mirailabs-co/ui-kit-shards-tech

Usage​

Parameters 
interface UIKitProps {
	accessToken: string;
	clientId: string;
	env?: string;
}

  • accessToken: user token for verification with Shards. Details: Authentication

  • clientId: ID of the game (provided when integrating with Shards)

  • env (default: development): environment of the game

How to import 
import UIKitShardsTech from "@mirailabs-co/ui-kit-shards-tech"

...
    const accessToken = `USER_ACCESS_TOKEN`;
    const clientId = `GAME_CLIENT_ID`;
    const env = 'production';


    ...
        <UIKitShardsTech
            accessToken={accessToken}
            clientId={clientId}
            env={env}
        >
