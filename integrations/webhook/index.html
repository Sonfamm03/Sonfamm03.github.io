Webhook (optional)

Partner can optionally setup an API Webhook to receive guild-related information from our smart contract.

  • method: POST
  • body:
{
  "key": "Name Data Event",
  "value": "Data Event"
}

          {
            key: 'newGuild',

              metadata: action.metadata,

                userId: userId,
                address: transaction.from,
              },
            }),

Change Owner Guild​

{
            key: 'changeOwnerGuild',
            value: JSON.stringify({
              _id: guildRecord._id.toString(),
              guildAddress: guildRecord.address,
              oldOwner: {
                userId: oldUser?.userId,
                address: oldUser?.address,
                clientId: oldUser?.clientId,
              },
              newOwner: {
                userId: userNewOwner?.userId,
                address: userNewOwner?.address,
                clientId: userNewOwner?.clientId,
              },
            }),
          }

Disband Guild​

{
            key: 'disbandGuild',
            value: JSON.stringify({
              _id: guildRecord._id.toString(),
              address: event.data.guild,
              name: guildRecord.name,
              users: [],
              clientId: guildRecord.clientId,
              metadata: guildRecord.metadata,
              owner: null,
            }),
          }
