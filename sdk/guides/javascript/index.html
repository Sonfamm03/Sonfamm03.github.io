Before you begin​

Shards Tech SDK​

npm install @mirailabs-co/shards-tech

yarn add @mirailabs-co/shards-tech

Leaderboards​

getLeaderBoards(): Promise<import("../transports/dtos/LeaderBoard.dtos").LeaderBoards[]>;
  • Fetche the list of leaderboards.

Guild Scores​

getGuildScores(query: GetGuildScoreDto): Promise<import("../transports/dtos/GuildScore.dtos").GuildScore[]>;
  • Fetche guild scores based on leaderboard and optional parameters.

Transaction History​

getTransactionHistoryOfUser(query: GetTransactionHistoryDto): Promise<import("../transports/dtos/TransactionHistoty.dto").TransactionHistory[]>;
  • Fetche user transaction history.
getTransactionHistoryOfGuild(query: GetTransactionHistoryDto): Promise<import("../transports/dtos/TransactionHistoty.dto").TransactionHistory[]>;
  • Fetche guild transaction history.

Join Guild Request (If game set user must request to join guild)​

createJoinGuildRequest(guildId: string): Promise<JoinGuildRequest>;

getJoinGuildRequest(guildId: string): Promise<JoinGuildRequest[]>;

getJoinGuildOfUser(accessToken: string): Promise<JoinGuildRequest[]>;
  • Create and get the list join-guild request

Guild Chat​

getGuildChatHistory(query: GetHistoryChatDto): Promise<GuildHistoryChat[]>;

getUserOnlineInGuild(): Promise<string[]>;

sendMessage(body: SendMessageDto): Promise<GuildHistoryChat>;
  • Guild Chat function

Guild Fraction and Slot Information​

getMyFractions(): Promise<Guilds[]>;
getTotalFractionOfGuild(guildId: string): Promise<number>;
getMyFractionOfGuild(guildId: string): Promise<number>;
getBuySlotPrice(guildId: string): Promise<string>;
getBuyFractionPrice(guildId: string, amount: number): Promise<string>;
getSellFractionPrice(guildId: string, amount: number): Promise<string>;
  • Fetche user's fractions, total fractions of a guild, fraction price, sell fraction price, and user's fraction of a guild, respectively.

Guild Creation​

createGuild(name: string, slotPrice: number, txGuildOwnerShare: number, rewardShareForMembers: number, guildOwnerShare: number, metadata: Record<string, any>): Promise<import("../transports/dtos/Action.dtos").Action>;
  • Create a new guild with the specified parameters.

Slot and Fraction Transactions​

buySlot(guildAddress: string): Promise<import("../transports/dtos/Action.dtos").Action>;
burnSlot(guildId: string): Promise<boolean>;
linkAddress(): Promise<import("../transports/dtos/Action.dtos").Action>;

// User sell seat
sellSlot(guildAddress: string, price: number): Promise<UserSellSlot>;
updateSellSlot(sellSlotId: string, price: number): Promise<UserSellSlot>;
cancelSellSlot(sellSlotId: string): Promise<boolean>

// Fractions/Shards
buyFraction(guildAddress: string, amount: number): Promise<import("../transports/dtos/Action.dtos").Action>;
sellFraction(guildAddress: string, amount: number): Promise<import("../transports/dtos/Action.dtos").Action>;
  • Perform various seats and fractions transactions.
