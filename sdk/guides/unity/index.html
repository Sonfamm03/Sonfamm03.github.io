Unity SDK

Overview​

The ShardsTech API provides functionality for integrating with the ShardsTech platform, allowing users to interact with various features such as guild management, user authentication, transaction history, chat functionality, referral systems, and more.

Package Dependencies​

  • "com.unity.nuget.newtonsoft-json"
  • "com.itisnajim.socketiounity"

Initialize​

  • Configure MiraiSDKSettings ClientID and AppSchema
  • Call ShardsTech.InitAsync if you don't use Auto Init option.
public static async Task InitAsync()

Authentication​

  • When user log-in or log-out your system, please call ShardsTech.Login / ShardsTech.Logout;
public static async Task Login(string token)
public static void Logout()
``
- Example:
```csharp
private async void Start()
{
    await MiraiSDK.InitAsync();
    GameAuth.OnLoginStateChanged += OnAuthStateChanged;
    OnAuthStateChanged(GameAuth.AccessToken);
}
private void OnAuthStateChanged(string token=null)
{
    if (string.IsNullOrEmpty(token)) ShardsTech.Logout();
    else _ = ShardsTech.Login("Bearer " + token);
}

Class: ShardsTech​

The ShardsTech class encapsulates functionality for interacting with the ShardsTech API.

Properties​

  • ShardsTechConfig: Configuration settings for ShardsTech.
  • MyUser: Information about the current user.
  • MyGuild: Information about the guild associated with the current user.
  • MySeatOnSale: Information about the Seat currently on sale by the user.
  • MeIsOwner: Indicates whether the current user is the owner of the guild.
  • IsLinkedAddress: Indicates whether the current user has a linked address.

Methods​

  • InitAsync(): Initializes the ShardsTech API asynchronously.
  • GetLeaderboards(): Retrieves leaderboard data.
1 #region User
  • Login(string token): Logs in the user using the provided authorization header.
  • Logout(): Logs out the current user.
  • LinkAddress(CancellationToken cancellationToken = default): Links the user's address.
  • FetchMyUser(): Retrieves information about the current user.
  • GetUserHistories(int page = 1, int limit = 100): Retrieves user transaction history.
  • GetUserScores(string leaderBoardId, int page = 1, int limit = 100, SortType sort = SortType.desc): Retrieves user scores for a specific leaderboard.
2 #region Guild
  • CreateGuild(string name, double seatPrice, object metadata, float txGuildOwnerShare, ProfitPercentConfig profitPercentConfig, CancellationToken cancellationToken = default): Creates a guild.
  • GetGuildScores(string leaderBoardId, string name = "", int page = 1, int limit = 100, SortType sort = SortType.desc): Retrieves guild scores for a specific leaderboard.
  • GetGuildHistories(int page = 1, int limit = 100): Retrieves guild transaction history.
  • GetUsersOfGuild(string guildId): Retrieves users belonging to a guild.
2.1 #region MyGuild
  • FetchMyGuild(): Retrieves information about the guild associated with the current user.
  • UpdateGuild(string name, double seatPrice, float txGuildOwnerShare, ProfitPercentConfig profitPercentConfig, CancellationToken cancellationToken = default): Updates guild information.
  • UpdateLogoDescriptionGuild(string avatar, string description, CancellationToken cancellationToken = default): Updates guild logo and description.
2.1.1#region Chat
  • OnlineUsersMyGuild(): Retrieves online users in the current guild.
  • GetChatHistory(int page = 1, int limit = 100): Retrieves chat history.
  • SendChat(string message): Sends a chat message.
2.2 #region JoinGuildRequest
  • CreateJoinGuildRequest(string guildId): Creates a join guild request.
  • GetJoinGuildRequestsOfUser(): Retrieves join guild requests of the current user.
  • GetJoinGuildRequestsOfMyGuild(): Retrieves join guild requests for my guild.
  • GetJoinGuildRequests(string guildId): Retrieves join guild requests for a specific guild.
  • Accept(string guildId, string userId, CancellationToken cancellationToken = default): Accept join guild request.
  • Reject(string guildId, string userId, CancellationToken cancellationToken = default): Reject join guild request.
3 #region Fractions
  • GetMyFractions(): Retrieves the current user's Fractions.
  • GetTotalFractionsOfGuild(string guildId): Retrieves the total fraction of a guild.
  • GetMyFractionsOfGuild(string guildId): Retrieves the user's fraction of a guild.
  • GetBuyFractionsPrice(string guildId, long amount): Retrieves the buy fractions price.
  • GetSellFractionsPrice(string guildId, long amount): Retrieves the sell fractions price.
  • BuyFractions(string guildAddress, long amount, long index = 0, CancellationToken cancellationToken = default): Buys Fractions from a guild.
  • SellFractions(string guildAddress, long amount, CancellationToken cancellationToken = default): Sells Fractions to a guild.
4 #region Seat
  • GetSeatsOnSale(string guildId = null): Retrieves seats on sale.
  • GetBuySeatPrice(string guildId): Retrieves the buy seat price.
  • BuySeat(string guildAddress, string seller, double price, CancellationToken cancellationToken = default): Buys a seat from a user.
  • SellSeat(double price, CancellationToken cancellationToken = default): Sells a seat.
  • FetchMySeatOnSale(): Retrieves information about the seat currently on sale by the user.
  • UpdateSellSeatPrice(string sellSeatId, double price, CancellationToken cancellationToken = default): Updates the price of a seat on sale.
  • CancelSellSeat(string sellSeatId, CancellationToken cancellationToken = default): Cancels the sale of a seat.
  • BurnSeat(CancellationToken cancellationToken = default): Burns a seat.

This draft provides an overview of the classes, properties, and methods available in the ShardsTech API.
