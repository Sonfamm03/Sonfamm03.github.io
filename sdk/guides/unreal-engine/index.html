Unreal Engine

Quickstart​

Import and configure the Unreal Engine SDK for your project.

Initialize​

ShardsTech::Init("clientID",[](bool success){
    UE_LOG(LogTemp,Display,TEXT("Init done %d"),success);
});

Authentication​

  • When you login/logout your system please call ShardsTech::Login / ShardsTech::Logout;
ShardsTech::Login("token",[](bool success)=>{
    UE_LOG(LogTemp,Display,TEXT("Login done %d"),success);
});
ShardsTech::Logout();

Class: ShardsTech​

The ShardsTech class encapsulates functionality for interacting with the ShardsTech API.

Properties​

  • ShardsTechConfig: Configuration settings for ShardsTech.
  • MyUser: Information about the current user.
  • MyGuild: Information about the guild associated with the current user.
  • MySlotOnSale: Information about the slot currently on sale by the user.
  • MeIsOwner: Indicates whether the current user is the owner of the guild.
  • IsLinkedAddress: Indicates whether the current user has a linked address.

Methods​

  • Init(): Initializes the ShardsTech API asynchronously.
  • GetLeaderboards(): Retrieves leaderboard data.
1 #region User
  • Login(string token): Logs in the user using the provided authorization header.
  • Logout(): Logs out the current user.
  • LinkAddress(): Links the user's address.
  • FetchMyUser(): Retrieves information about the current user.
  • GetUserHistories(): Retrieves user transaction history.
  • GetUserScores(): Retrieves user scores for a specific leaderboard.
2 #region Guild
  • CreateGuild(): Creates a guild.
  • GetGuildScores(): Retrieves guild scores for a specific leaderboard.
  • GetGuildHistories(): Retrieves guild transaction history.
  • GetUsersOfGuild(): Retrieves users belonging to a guild.
2.1 #region MyGuild
  • FetchMyGuild(): Retrieves information about the guild associated with the current user.
  • UpdateGuild(): Updates guild information.
  • UpdateLogoDescriptionGuild(): Updates guild logo and description.
2.1.1#region Chat
  • OnlineUsersMyGuild(): Retrieves online users in the current guild.
  • GetChatHistory(): Retrieves chat history.
  • SendChat(): Sends a chat message.
2.2 #region JoinGuildRequest
  • CreateJoinGuildRequest(): Creates a join guild request.
  • GetJoinGuildRequestsOfUser(): Retrieves join guild requests of the current user.
  • GetJoinGuildRequests(): Retrieves join guild requests for a specific guild.
3 #region Share
  • GetMyShares(): Retrieves the current user's shares.
  • GetTotalShareOfGuild(): Retrieves the total share of a guild.
  • GetMyShareOfGuild(): Retrieves the user's share of a guild.
  • GetBuySharePrice(): Retrieves the buy share price.
  • GetSellSharePrice(): Retrieves the sell share price.
  • BuyShare(): Buys shares from a guild.
  • SellShare(): Sells shares to a guild.
4 #region Slot
  • GetSlotsOnSale(): Retrieves slots on sale.
  • GetBuySlotPrice(): Retrieves the buy slot price.
  • BuySlot(): Buys a slot from a user.
  • SellSlot(): Sells a slot.
  • FetchMySlotOnSale(): Retrieves information about the slot currently on sale by the user.
  • UpdateSellSlotPrice(): Updates the price of a slot on sale.
  • CancelSellSlot(): Cancels the sale of a slot.
  • BurnSlot(): Burns a slot.

This draft provides an overview of the classes, properties, and methods available in the ShardsTech API.
